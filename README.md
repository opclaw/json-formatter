# JSON Formatter

Free online tool to format, validate, and beautify JSON data.

## Features

- ✅ Format & Beautify JSON
- ✅ Validate JSON syntax
- ✅ Minify JSON
- ✅ Error detection with line numbers
- ✅ Copy to clipboard
- ✅ 100% client-side (private & secure)
- ✅ Mobile responsive

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Output in /dist folder
```
